## Instalación

```bash
npm install
```
## Pruebas

```bash
npm test
```


## Implementaciones
- Flexbox
- CSS Grid
- SCSS
- Responsive hasta 300px
- Linter [ESLint](https://eslint.org/)
- React
- Hooks
- LocalStorage
- Redux
- Pruebas con Jest, react testing, redux mock store
- Lazy loading con React.Lazy y React.Suspense
- Desplegado en [netlify](https://www.netlify.com/)
